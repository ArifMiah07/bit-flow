## 📄 Overview

This is a responsive, single-page React application showcasing the design, services, and team section of BIT FLOW. It is inspired by a modern Dribbble design and customized to reflect branding, simplicity, and usability.

Technologies used:
- React + Vite
- React Router
- Tailwind CSS

## 📊 Analysis

- Fully responsive layout using **Tailwind CSS**
- Clean and minimal **component structure**
- Client-side routing with **React Router**
- Two additional custom pages added: **About Us** and **Our Works**
- Deployed on **Vercel** with fast load performance and zero downtime

## ⚙️ Run and Deploy

### Local Setup

```bash
git clone https://github.com/ArifMiah07/bit-flow.git
cd bit-flow
npm install
npm run dev
```

### Deploy

This project is deployed on Vercel:
